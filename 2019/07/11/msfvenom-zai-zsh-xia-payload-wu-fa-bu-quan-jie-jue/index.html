msfvenom在zsh下payload无法补全解决

前期准备

首先要想msfvenom有代码提示，需要安装zsh

sudo apt-get install zsh

然后安装oh-my-zsh

sh -c "$(wget https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"

安装脚本

git clone https://github.com/Green-m/msfvenom-zsh-completion ~/.oh-my-zsh/custom/plugins/msfvenom/

查看是否下载成功

ls ~/.oh-my-zsh/custom/plugins/msfvenom/

修改配置文件

vi ~/.zshrc #找到plugins=(git),改成plugins=(msfvenom)
#下面两条添加到最底下
fpath=(~/.zsh/completion$fpath) #（$fpath 通过添加来包含您的目录~/.zshrc
autoload-Uzcompinit&&compinit-i #（确保 compinit 已加载或通过添加~/.zshrc）

保存退出，执行

source ~/.zshrc

问题解决

发现其参数都可以智能提示，但是payload没有。看了下源码，找到一种解决方案

方法就是把payload添加到~/.zsh/venom-cache这个文件中下次再使用就有提示了。
